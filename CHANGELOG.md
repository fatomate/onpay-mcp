# Changelog

All notable changes to `@fatomate/onpay-mcp` are documented here.

## [0.1.2] - 2026-06-11

- Add a `.mcpb` one-click bundle for Claude Desktop.
- README badges (npm version, license).

## [0.1.1] - 2026-06-11

- First public release on npm.
- Three tools over stdio: `sales_list`, `sales_get`, `sales_update_shipping_info`.
- Support both `ONPAY_ACCOUNT` (subdomain) and `ONPAY_BASE_URL` (custom domain).
